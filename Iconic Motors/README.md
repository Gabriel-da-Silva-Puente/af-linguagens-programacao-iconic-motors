# Iconic Motors

## Sobre o Projeto

O Iconic Motors é um site com tema de museu virtual de carros e o objetivo é apresentar carros icônicos, novidades do mercado e informações sobre veículos.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## Páginas do Projeto

### Home

Página inicial com carrossel e cards de carros em destaque

### Galeria

Página com carros divididos em categorias como novidades, ícones e lendas

### Contato

Página com formulário para contato e mapa de localização
